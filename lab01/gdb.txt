1. Set program arguments:
   set args <arguments>
2. Create a breakpoint:
   break <function_name> or break <line_number>
3. Execute next line of C code:
   next
4. Step into a function:
   step
5. Continue after breakpoint:
   continue
6. Print value of variable/expression:
   print <variable/expresson>
7. Display variable value after every step:
   display <variable_name>
8. Show all variable in current function:
   info locals
9. Quit GDB:
   quit
